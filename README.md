# VELUGEETHA – Special Education Academic Ebooks

This repository hosts resources and documentation for the **VELUGEETHA** initiative, 
focused on creating and distributing multilingual academic ebooks in the field of special education.

## 📖 About the Project
VELUGEETHA aims to make academic content accessible across languages, 
supporting educators, learners, and institutions with affordable digital resources.

## ✨ Features
- Six individual ebooks across multiple languages
- One bundled collection for comprehensive learning
- Standardized pricing with Razorpay integration
